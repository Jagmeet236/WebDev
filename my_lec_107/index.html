The .box1 class uses the default box model (content-box), where the width and height properties include only the content of the box, and the padding and border are added to the outside of the box

The .box2 class uses the border-box model, where the width and height properties include the content, padding, and border. This means the size of the box remains the same regardless of padding or border thickness.

Margin Collapse

The top and bottom margins of blocks are sometimes combined (collapsed) into a single margin whose size is the largest of the individual margins or just one of them, if they are equal.

Total Width And Height

Total Width = Width + Left Padding + Right Padding + Left Border + Right Border + Left Margin + Right Margin
Total Height = Height + Top Padding + Bottom Padding + Top Border + Bottom Border + Top Margin + Bottom Margin
